> منصة الدعم النفسي والصحة العقلية - مساحة آمنة لمشاركة الأفكار والمشاعر

**Khawater** est une plateforme de santé mentale et de soutien psychologique qui offre un espace sûr et bienveillant pour partager ses pensées, émotions et expériences. Conçue avec une approche respectueuse de la culture arabe et des besoins en santé mentale.

## ✨ Fonctionnalités

### 🎯 **Fonctionnalités Principales**
- **Posts anonymes ou identifiés** - Partagez en toute sécurité
- **Système d'humeurs** - 6 états émotionnels avec couleurs associées
- **Interface responsive** - Optimisée mobile-first
- **Support RTL** - Interface native en arabe
- **Temps réel** - Synchronisation instantanée
- **PWA Ready** - Installable comme app mobile

### 🎨 **Humeurs Disponibles**
- 😌 **Emerald** - مسالم (Paisible)
- 😊 **Amber** - سعيد (Heureux)  
- 😢 **Ocean** - حزين (Triste)
- 😰 **Sunset** - قلق (Anxieux)
- 😇 **Purple** - هادئ (Calme)
- 🤗 **Rose** - داعم (Soutenant)

### 🔒 **Sécurité & Confidentialité**
- Chiffrement des données sensibles
- Mode anonyme complet
- Pas de tracking personnel
- Données stockées de manière sécurisée
- Conformité RGPD

## 📦 Installation

### **Prérequis**
- Navigateur moderne (Chrome, Firefox, Safari, Edge)
- Connexion Internet
- Git (pour le développement)

### **Installation Rapide**

```bash
# Cloner le repository
git clone https://github.com/votre-username/khawater.git
cd khawater

# Option 1: Serveur local simple
python -m http.server 8000
# ou
npx serve .

# Option 2: Live Server (VS Code)
